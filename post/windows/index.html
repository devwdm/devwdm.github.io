UEFI引导修复

如果你以任何方式都无法进入 Windows 10，则需要一些安装盘来启动，去修复：安装盘启动修复界面，按 Shift + F10，打开终端，输入 DISKPART

$ list disk
$ select disk n           # (n) 为你 Windows 10 所在的磁盘
$ list partition
$ select partition n           # (n) 为你 Windows 10 所在硬盘的 EFI 分区
$ assign letter=b         # 将 EFI 分区设为 b 盘
$ exit
$ cd /d b:\EFI\Microsoft\Boot\
$ bootrec /fixboot
#如果通过其他方式能进入 Windows 10，则可以打开管理员权限的终端

$ diskpart
$ list disk
$ select disk n
$ list partition
$ select partition n
$ assign letter=b:
$ exit
$ cd /d b:\EFI\Microsoft\Boot\
$ bcdboot c:\Windows /l zh-CN /s b: /f ALL
#重启电脑ok

Windows与Mac时间不一致

解决方式执行以下cmd命令

$ Reg add HKLM\SYSTEM\CurrentControlSet\Control\TimeZoneInformation /v RealTimeIsUniversal /t REG_DWORD /d 1
