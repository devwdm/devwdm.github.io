光猫吉比特GS3101改为桥接路由器拨号模式

2020-07-08 17:07:00

光猫吉比特GS3101改为桥接路由器拨号模式

  1. 为什么要这么干呢？（备注：亲戚家的网速为300M，当网速小于100M就没有必要折腾了）
    1. 因为这个光猫不满足我的使用要求
    2. 没有5G频段的WiFi，旧WiFi频段速度太慢不到100M
    3. 不稳定
    4. 经常掉线
    5. 看视频缓冲速度慢。
  2. 这样干的好处
    1. 现有资源最优化
    2. 网速更快，网上冲浪时减少等待时间；不浪费时间在等待上。
    3. 不被光猫的性能限制网速
    4. 让光猫只干一件事情，光电转化。
    5. 把你的宽带速率提升到所开通的业务速率（300M）
  3. 有什么缺点
    1. 费钱，需要自己买一个路由器
    2. 折腾，需要一点点的技术
    3. 暂时想不到了

一、获取超级管理员账号密码

windows 使用cmd命令

mac 先安排telent工具，

$ brew install telent

通过telnet工具进入后台

$ telnet 192.168.1.1
Trying 192.168.1.1...
Connected to 192.168.1.1.
Escape character is '^]'.
tc login: admin
Password: s2@We3%Dc#

cd 切换路径

$ cd /tmp
# 查看用户名
$ cat ctromfile.cfg |grep 'user'

# 查看密码
$ cat ctromfile.cfg |grep 'passw'

然后就可以通过web 192.168.1.1登录调整相应的参数了。
